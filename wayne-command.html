Integration guide
Notion — Operation Mandate todos
01
Create a Notion integration
Go to notion.so/my-integrations → New integration → give it a name like "Wayne Command" → copy the Internal Integration Token.
https://www.notion.so/my-integrations
02
Connect it to your Operation Mandate page
Open your Operation Mandate Notion page → ··· menu (top right) → Connections → add your integration. This gives it read/write access to that page only.
03
Get your database ID
Open the database in Notion in your browser. The URL looks like: notion.so/[workspace]/[DATABASE_ID]?v=... — copy the 32-character ID between the last / and the ?
04
Add credentials to this file
In the HTML file, find the JS section labelled NOTION CONFIG and paste in your token and database ID. The Pull/Push buttons will then call the API directly.
const NOTION_TOKEN = 'secret_xxx...';
const NOTION_DB_ID = 'your-db-id-here';
⚠
CORS limitation
Notion's API blocks direct browser calls. You need either: (a) a free Cloudflare Worker as a proxy — takes 10 min to set up, or (b) use Claude.ai with the Notion MCP connector already linked to your account — which is what the Sync buttons currently guide you to do.
Google Calendar
01
Enable Google Calendar API
Go to console.cloud.google.com → New project → Enable Google Calendar API → Create credentials (OAuth 2.0 Client ID for a web app).
https://console.cloud.google.com/apis/library/calendar-json.googleapis.com
02
Get an API key for read access
For read-only (just displaying events): Create credentials → API key → restrict it to Calendar API. Add it to the HTML under GCAL CONFIG. Your calendar must be set to public or you must use OAuth.
const GCAL_API_KEY = 'AIza...';
const GCAL_ID = 'your@gmail.com';
03
Fetch events
Once configured, the Agenda tab will pull live events from your Google Calendar using the public API endpoint. Tell me when you have the API key and I'll wire it up.
https://www.googleapis.com/calendar/v3/calendars/{id}/events?key={key}
Portfolio tracker
01
Currently: iframe embed
The Portfolio tab iframes your existing tracker at spaniecom-source.github.io/PortofolioTracker — it's fully live. The Overview panel pulls live Yahoo Finance prices separately for the snapshot widget.
02
To get real P&L on the Overview
Update your actual share counts in the STOCKS array in the JS. Find "const STOCKS = [" and edit the shares: value for each ticker to match your real positions. The prices are already live from Yahoo Finance.
{ ticker:'NVDA', name:'Nvidia', shares: 5, color:'#4caf7d' }
